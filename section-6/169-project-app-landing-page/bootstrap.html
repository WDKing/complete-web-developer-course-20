How We Develop

Smallest Computers

We develop this application on the smallest computers. Apple Mac laptops are fairly common when developing.

Multiple Monitors

We attach our smallest computers to the largest monitors, so that we can see what we are doing.

Version Control

Always make sure that you have a backup! Version control lets multiple developers work on different projects concurrently, and you can always get back what was lost. It is not a matter of if, but when!
